# Wordly Dictionary SPA

A Single Page Application (SPA) built using HTML, CSS, and JavaScript.

## Features
- Search for words
- View definitions, examples, and synonyms
- Pronunciation audio
- Favorites list
- Error handling

## API Used
Free Dictionary API

## How it works
Users type a word, and the app fetches data from an external API and displays it dynamically without refreshing the page.

## Technologies
- HTML
- CSS
- JavaScript (Fetch API)
